HTML Code Sheet

A look inside HTML basic tags, table tags, media tags and semantic tags.


Basic HTML Tags

Tag Name Attributes Description
<html> HTML Defines the root of an HTML document
<head> Head title, meta, style, link, script Contains metadata/information for the document
<body> Body Defines the document's body
<p> Paragraph Defines a Paragraph
<h1> to <h6> Heading Define a HTML heading. <h1> defines the most important. <h6> defines the least important
<div> Division A container that divides the page into sections
<a> Anchor href,target Used to link to other locations. Uses attribute href="" to specifies the link's destination and target="_blank" to open new websites in a new tab or window
<ul> Unordered List Defines an unordered list
<ol> Ordered List Defines an ordered list
<li> List Item Defines a list item
<!--...--> Comment Defines a comment. It doesn't appear in the browser.
<span> Span Defines a section in a document. This tag tag is an inline container used to mark up a part of a text, or a part of a document
<strong> Strong Used to define text with strong importance. The content inside is typically displayed in bold.
<em> Emphasize Used to define emphasized text. The content inside is typically displayed in italic.
<br> Break Defines a single line break
<button> Button Defines a clickable button
<abbr> Abbreviation defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM"
<blockquote> Quote Specifies a section that is quoted from another source


Table HTML Tags

Tag Name Attributes Description
<table> Table Defines a table
<thead> Table Head Groups the header content in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row Defines an entire row of a table
<th> Table Head colspan, rowspan Usually the column heads in a table. Renders bold
<td> Table Data colspan, rowspan Cell content
<tfoot> Table Foot Groups the footer content in a table
<caption> Caption Defines a table caption. Must be inserted immediately after the <table> tag


Media HTML Tags

Tag Name Attributes Description
<img> Image src, alt Adds an image to the document. Self-closing tag
<video> Video src, controls, autoplay, loop Display a video to the user
<audio> Audio controls, autoplay Embend audio content into a document
<source> Source src, type="audio/mp3" Adds src to audio. Used inside the <audio> tag
<embed> Embed Incorporate any media content to a page (videos, audio files, and gifs from external source. It's a self-closing tag


Semantic HTML5 Tags

Tag Name Attributes Description
<header> Header Used for either navigational links or introductory content containing <h1> to <h6> headings
<nav> Navigation Defines a block of navigation links such as menus and table of contents
<main> Main Defines content of a page
<section> Section Defines elements in a documentm such as chapters, headings, or any other area of the document with the same theme
<article> Article Holds content that makes sense on its own. Helps someone using a screen reader understand where the article content begins and ends
<figure> Figure Used to encapsulate media such as an image, illustration, diagram, code snippet, etc
<figcaption> Figure Caption Used to describe the media in the <figure> tag
<aside> Aside Used to mark additional information that can enhance another element but isn't required in order to understand the main content.
Common uses of the <aside> element include: Bibliographies, Endquotes, Comments, Pull Quotes, Editorial sidebars, Additional Information
<footer> Footer The content at the bottom of the subject information.
Usually used for Contact Information, Copyright Information, Terms of Use, Site Maps and Reference to Top of Pages Links
